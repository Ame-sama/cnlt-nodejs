# Bài Thực Hành Số 3 - NodeJS & ExpressJS

Dự án này sử dụng Node.js và framework Express để xây dựng một Web App hoàn chỉnh, tích hợp giao diện chuyên nghiệp từ template Clean Blog (Start Bootstrap).

## Các chức năng chính
- Trang chủ (Home): Giao diện Bootstrap.
- Trang Danh sách (/list): Hiển thị danh sách  bằng vòng lặp EJS.
- Trang Chi tiết (/detail/:id): Sử dụng Dynamic Routing để hiển thị thông tin.
- Trang Liên hệ (/contact)

## Công nghệ sử dụng
- Backend: Node.js, Express.js.
- Frontend: EJS Template Engine, Bootstrap.
